# js-tx-example

### Requirements

- node.js v18.17.1
- yarn v1.19.2

### How to run project

1. Install dependencies:
   ```bash
   yarn
   ```

2. Check the patch files created with the `postinstall` script

3. Add `.env` file following the `.evn.example`

4. Run example code
   ```bash
   yarn start
   ```
